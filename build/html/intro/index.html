1. 主要功能特性¶

  1. 用于模型在环仿真（MiL）的C/C++代码集成工具

  2. 简捷、统一，一次编码同时支持FMI及S-Function模型接口标准

  3. 使用CMake构建系统，可自动适配大多数编译器

  4. 使用Qt图形界面，可运行于大部分操作系统的桌面环境

  5. 与来自IBK的MasterSimulator环境集成发布，便于FMU集成仿真

  6. 生成平台相关的代码，兼容Windows及Linux

  7. 生成的模型可运行在基于Linux的半实物仿真（HiL）系统
